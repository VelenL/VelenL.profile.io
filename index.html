Anomaly Detection – Breast Cancer

Developed an unsupervised anomaly detection pipeline using the Multivariate Gaussian model to identify rare malignant cases in a breast cancer dataset. Optimized the threshold (epsilon) by maximizing F1-score to improve detection accuracy. Visualized detected outliers that deviated from the main cluster, revealing potential malignancies.

Python Matplotlib Gaussian Model Anomaly Detection

Book Recommendation System

Built a content-based recommender using the Goodreads “Best 1500 Books” dataset. Combined metadata into textual features and applied CountVectorizer with Cosine Similarity to suggest similar books. Added a popularity-based ranking using average rating and number of raters.

CountVectorizer Cosine Similarity Recommender System

Time Series — Marseille Airport Traffic Forecasting

Built a seasonal forecasting model for Marseille airport’s monthly passenger traffic using log transformation, differencing, and SARIMA/SARIMAX. Conducted residual diagnostics and outlier corrections to ensure model stability and interpretability. The final model successfully captured long-term trends and seasonality, producing accurate forecasts for operational planning.

R Time Series SARIMA Forecasting

International Student Support Chatbot

Developed an intent-based chatbot using Python, Flask, and Streamlit to assist international students with daily life topics. The system combines tokenization, lemmatization, stop-word removal, and bag-of-words vectorization to classify user questions through a neural network (Keras). The trained model is served via a Flask API and connected to an interactive Streamlit web interface for real-time responses.

Python Flask Streamlit Keras NLP Chatbot
